# humanities-doc-toolkit
面向人文学科研究与文献治理的命令行工具链：批量提取文献信息并重命名、低风险可复核的智能归档分类（dry-run + apply）、按研究主题智能分拣；支持多线程与多密钥轮询。Humanities Doc Toolkit (HDT) — a CLI toolchain for humanities literature management: batch renaming via metadata extraction, auditable low-risk classification (dry-run + apply), and topic-based sorting; supports multi-thread &amp; multi-key rotation.
